# 🧪 ChemLab Pro Matrix (High-Precision Calculation Matrix for Chemistry and Polymers)
👉 **[Click here to access and use the calculator directly (https://chemcale.streamlit.app/)]**

## 📖 Project Introduction
This project is specifically developed for chemistry and polymer laboratories, aiming to provide a simple, high-precision, and mobile-enabled central hub for feed and solution calculations. The program features a three-decimal-place (0.001) high-precision mode and operates strictly based on `mol` and `g` logic, completely eliminating unit conversion hell.

## ✨ Core Functional Modules
### 1. 🔗 Polymer Specialist
- Designed specifically for controlled polymerization reactions such as ATRP, RAFT, and ROP.
- Automatically calculates the initiator and catalyst feed amounts by setting the target degree of polymerization (DP) of the monomer.
- Supports real-time calculation of the final monomer concentration $[M]_0$.
### 2. 💧 Stock Solutions
- A must-have tool for pre-production preparation.
- Input target concentration and volume, and it automatically distinguishes between solid and liquid reagents based on their density.
- Accurately outputs the solid weighing mass (mg) or liquid transfer volume (μL) and the volume of pure solvent to be added.
### 3. ⚖️ Stoichiometry
- A powerful tool for small molecule organic synthesis calculations.
- Supports freely setting reaction baselines (limited reagents), and calculates the amount of all reactants and the theoretical yield of products with a single click using equivalence (eq).
- Built-in reagent purity (%) compensation calculation and intelligent mass-volume conversion tool.
